## 🚀 How to Run

1. Clone the repo:

   git clone https://github.com/Aadya2901/codeforces-grind.git
   cd codeforces-grind


2. Compile & run a solution:

   g++ A_Level/231A_Team.cpp -o team
   ./team

Or use a C++ IDE like VS Code, CodeBlocks, or Sublime with C++ extension.
